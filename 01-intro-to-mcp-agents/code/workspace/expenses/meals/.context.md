# Meals Expense Specifics

Meals expenses have additional requirements beyond standard expenses.

## Additional Required Fields
- **Location**: City and state/country
- **Time**: morning, noon, afternoon, evening
- **Number Participants**: 

## Naming Convention Override
Meals expense files should include destination:
`YYYY-MM-DD_location_#participants.md`

Example: `2024-01-15_chicago_4.md`
